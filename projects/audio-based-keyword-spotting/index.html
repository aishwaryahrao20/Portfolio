Audio-Based Keyword Spotting System Using TinyML

A low-latency, MFCC-based keyword spotting pipeline designed for embedded deployment and real-time on-device inference.

Overview

This project focused on building a complete TinyML workflow — from data collection and preprocessing to model optimization and deployment on an embedded platform.

What I built

  • Built a 250+ sample dataset and developed a low-latency MFCC-based classifier optimized for embedded deployment, enabling robust real-time inference.
  • Deployed the model on Arduino Nano 33 BLE Sense, demonstrating an end-to-end TinyML pipeline for on-device predictive analytics and automated event detection.

Engineering focus

  • Balanced latency, accuracy, and model size for resource-constrained hardware.
  • Showed a complete path from sensor input and feature extraction to embedded decision-making.
  • Designed the project to be clear, demonstrable, and deployment-oriented.
Project details
  • Duration: Aug 2025 – Dec 2025
  • Associated with: Arizona State University
  • Project type: Embedded AI / TinyML
