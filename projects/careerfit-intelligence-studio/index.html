CareerFit Intelligence Studio Job Match Intelligence

A Streamlit-based job intelligence platform that parses resumes and profile websites, fetches public jobs, and ranks opportunities with explainable match evidence.

View project report

Overview

CareerFit Intelligence Studio is a domain-agnostic job discovery and resume/profile intelligence platform. It converts uploaded resumes, public profile websites, and company career pages into ranked and explainable job matches through a Streamlit interface.

What I built

  • Implemented ATS connectors for Workday, Greenhouse, Ashby, Lever, SmartRecruiters, and custom public career pages.
  • Designed a Profile Studio, Source Hub, Job Command workflow, match cards, diagnostics, CSV export, and dark-mode/readability improvements.
  • Built an explainable scoring pipeline using title-role alignment, skill overlap, keyword overlap, role phrase matching, seniority adjustment, and negative-signal penalties.

Measured outcomes

  • Supported 6 job-source categories with connector badges and fallback crawling.
  • Used a configurable 85% high-fit threshold to prioritize strong matches.
  • Observed a local prototype scan that ranked 7 NVIDIA results, surfaced 2 high-fit cards, and completed in 2.5 seconds.
  • Bounded profile ingestion at up to 20 PDF/DOCX documents, 12 PDF pages per document, and 90,000 profile characters.

Engineering decisions

  • Used manual scan execution to keep compute intentional and visible.
  • Added 4-worker default parallel fetching, 6-hour disk caching, and fast Workday listing mode to improve responsiveness.
  • Separated true personalized relevance from no-profile fallback scores to avoid misleading match quality.
  • Designed around public job discovery and user-provided profile artifacts without candidate portal credential collection.
Project details
  • Project status: Working prototype
  • Interface: Streamlit
  • Connectors: Workday, Greenhouse, Ashby, Lever, SmartRecruiters, custom
  • High-fit threshold: 85% default
  • Observed scan: 7 ranked results, 2 high-fit cards, 2.5 seconds
