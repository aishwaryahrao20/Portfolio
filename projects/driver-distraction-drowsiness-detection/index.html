Driver Distraction & Drowsiness Detection using In-Vehicle Signal Processing

A multi-sensor monitoring system that uses anomaly detection, event signatures, and control logic to support fast, reliable response.

Overview

This project focused on extracting meaningful metrics from vehicle-related signals and translating them into a closed-loop alerting system for real-time monitoring.

What I built

  • Built a multi-sensor monitoring system using anomaly detection and control logic, extracting metrics such as PERCLOS, EAR, and gaze deviation for real-time pattern detection.
  • Implemented time-series analysis and a closed-loop alert mechanism triggered by event signatures, demonstrating reliable fault monitoring and automated response logic.

Engineering focus

  • Combined sensing, analytics, and control behavior inside one practical workflow.
  • Used event-driven logic to convert detection into action.
  • Built a system that can be explained clearly to both technical and non-technical audiences.
