Project Intern - Center for Artificial Intelligence and Robotics (DRDO)

Developed autonomous ground vehicle perception workflows using LiDAR, RGB-D camera, Radar, ROS2, Gazebo, rviz, tf2 transforms, and repeatable simulation environments.

Overview

This internship focused on multi-sensor perception for autonomous ground vehicles. The work combined sensor-stream integration, simulation environment setup, 3D visualization, and time-aligned data synchronization for reliable obstacle detection and environmental mapping.

What I did

  • Developed a multi-sensor fusion pipeline for autonomous ground vehicle object detection by integrating LiDAR with 360-degree field of view, RGB-D camera depth data, and Radar streams using ROS2 and rviz.
  • Built and validated custom UGV simulation environments in Gazebo with 3D robot models designed in Blender, enabling repeatable autonomous navigation testing across cluttered obstacle scenarios without physical hardware dependency.
  • Implemented ROS2 sensor data synchronization across LiDAR, camera, and radar using tf2 transforms and topic-level message filtering, enabling time-aligned fusion for obstacle detection and environmental mapping at 30 FPS camera input.

Engineering value

  • Shows hands-on robotics platform integration across simulation, sensors, transforms, and visualization.
  • Demonstrates ability to build repeatable validation workflows before relying on physical hardware.
  • Connects perception inputs to system-level reliability through synchronization and structured testing.
